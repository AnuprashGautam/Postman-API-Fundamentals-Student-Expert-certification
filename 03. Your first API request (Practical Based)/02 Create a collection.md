### __Task: Create a Collection__

Collections are places to organize your API requests in Postman. They serve as executable documentation of API endpoints.

#### __Steps to Create a Collection__

1. **Open Postman**: Navigate to the left pane.
2. **Create a Collection**: 
   - Click the plus ("+") icon or "Create a collection".
   - Select "Blank collection".
   - Name your new collection "Postman Library API v2".
